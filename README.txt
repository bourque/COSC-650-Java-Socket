COSC 650 Java Socket Project

How to use the program:

(1) Make a file name "ip.txt" in the working directory that contains a list of
    IP addresses (one per line).  For example:

    136.160.171.110
    74.125.29.103

(2) Compile the code:

    javac c650BourqueServer.java
    javac c650BourqueClient.java

(3) Start the Client program:

    java c650BourqueClient

(4) Enter a timeout for the Client when prompted
(5) Start the Server program:

    java c650BourqueServer

(6) Enter a timeout for the Server when prompted
(7) Connect to localhost port 80 on your local browser
